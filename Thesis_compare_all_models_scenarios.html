1 Purpose

This R Markdown file compares all models and all imbalance-handling scenarios used in the thesis:

  1. No SMOTE: models are trained directly after preprocessing.
  2. SMOTE: synthetic minority oversampling is applied only inside the training/resampling recipe.
  3. Class weights: the minority class receives larger weights during model training.

The main outcome variable is Biopsy, and the positive/minority class is assumed to be Yes.

For this dataset, PR-AUC, MCC, sensitivity, and specificity are more informative than accuracy because the outcome is strongly imbalanced.

# Helper function: source script only if needed
source_if_needed <- function(csv_file, script_file) {
  csv_path <- file.path(base_dir, csv_file)
  script_path <- file.path(base_dir, script_file)
  
  if (!file.exists(csv_path)) {
    if (!file.exists(script_path)) {
      stop(paste("Missing both", csv_file, "and", script_file))
    }
    cat("CSV not found:", csv_file, "\n")
    cat("Running script:", script_file, "\n")
    source(script_path, local = .GlobalEnv)
  } else {
    cat("Found CSV:", csv_file, "\n")
  }
}

6 Confusion matrices

This section reads prediction files from the three scenarios and computes confusion matrices for every model.

read_predictions <- function(file, scenario_name) {
  path <- file.path(base_dir, file)
  if (!file.exists(path)) return(NULL)
  read.csv(path) %>%
    mutate(
      scenario = scenario_name,
      Biopsy = factor(Biopsy, levels = c("No", "Yes")),
      .pred_class = factor(.pred_class, levels = c("No", "Yes"))
    )
}

pred_all <- bind_rows(
  read_predictions("predictions_no_SMOTE.csv", "No SMOTE"),
  read_predictions("predictions_SMOTE.csv", "SMOTE"),
  read_predictions("predictions_class_weights.csv", "Class weights")
)

if (!is.null(pred_all) && nrow(pred_all) > 0) {
  confusion_summary <- pred_all %>%
    group_by(scenario, model) %>%
    summarise(
      TN = sum(Biopsy == "No"  & .pred_class == "No"),
      FP = sum(Biopsy == "No"  & .pred_class == "Yes"),
      FN = sum(Biopsy == "Yes" & .pred_class == "No"),
      TP = sum(Biopsy == "Yes" & .pred_class == "Yes"),
      .groups = "drop"
    ) %>%
    mutate(
      sensitivity_manual = TP / (TP + FN),
      specificity_manual = TN / (TN + FP),
      precision_manual = TP / (TP + FP)
    )
  
  confusion_summary %>%
    arrange(scenario, model) %>%
    kable(digits = 4, caption = "Confusion matrix summary by model and scenario") %>%
    kable_styling(full_width = FALSE)
} else {
  cat("Prediction CSV files were not found. Confusion matrices cannot be shown.\n")
}

read_importance <- function(file, scenario_name) {

11 Final recommendation template

Use this paragraph in the thesis and adjust numbers after knitting the document:

In this study, four machine learning models were compared under three imbalance-handling scenarios: no resampling, SMOTE, and class weights. Since the outcome variable Biopsy was highly imbalanced, the comparison focused primarily on PR-AUC and MCC rather than accuracy. The results indicated that the best-performing model according to PR-AUC was LASSO Logistic Regression in the No SMOTE scenario. However, the final model choice should also consider sensitivity and specificity. In a medical classification problem, a model with reasonable sensitivity is important because false negatives may be more problematic than false positives. If the class-weight method gives similar performance to SMOTE, it may be preferred because it preserves the original data distribution and does not generate synthetic patient profiles.
